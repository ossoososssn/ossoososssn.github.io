Houdini 18 安装教程

Houdini18.5安装教程

1：运行houdini-18.5.563-win64-vc141程序，开始安装

2:点击next

3:点击I agree

4:点击next

5:点击next

6:根据需要，选择软件安装路径，然后点击next

7:点击Install

8:等待安装即可

9:完成后，点击finish

破解鼠标右键点击Houdini-18.5-Keygen-Win，以管理员身份运行

11:点击Patch License Daemon，可以看到sever name和sever

12:在开始菜单找到License Administrator并打开

13:打开后点击sever information，看到sever name和sever code

14:运行破解文件houdini18.5keygen

15:将对应server name和server code，复制到注册机中，然后点击Generate License

16:在Houdini License Administrator界面的菜单栏的File菜单下选Manual Enter Keys

17:弹出注册码输入窗口，共需要输入五行注册码。然后复制注册机生成的License Keys窗口中的前五行注册码，对应输入即可

破解完成，点击close，可以开始使用软件
